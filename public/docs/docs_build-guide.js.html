/**
 * @file Build Instructions
 * Complete guide for compiling ShippingManager CoPilot into a standalone Windows installer.
 *
 * ## Prerequisites
 *
 * ### Required Software
 *
 * 1. **Node.js 22+** (https://nodejs.org/)
 *    ```bash
 *    node --version  # Should be >= 22.0.0
 *    ```
 *
 * 2. **Python 3.10+** (https://www.python.org/)
 *    ```bash
 *    python --version  # Should be >= 3.10
 *    ```
 *
 * 3. **PyInstaller** (for Python → .exe compilation)
 *    ```bash
 *    pip install pyinstaller
 *    ```
 *
 * 4. **pkg** (for Node.js → .exe compilation)
 *    ```bash
 *    npm install -g pkg
 *    ```
 *
 * 5. **NSIS 3.x** (for installer creation)
 *    ```bash
 *    # Install via winget (recommended)
 *    winget install NSIS.NSIS
 *
 *    # Or download manually: https://nsis.sourceforge.io/
 *    ```
 *    - NSIS will be automatically added to PATH
 *
 * ### Python Dependencies
 *
 * ```bash
 * pip install pywin32 cryptography selenium pillow
 * ```
 *
 * ### Node.js Dependencies
 *
 * ```bash
 * npm install
 * ```
 *
 * ## Build Process
 *
 * ### Full Build (Recommended)
 *
 * This will compile everything and create the installer in one command:
 *
 * ```bash
 * npm run build:all
 * npm run build:installer
 * ```
 *
 * **Output:**
 * - `dist/ShippingManagerCoPilot-Setup-v0.1.0.exe` (installer)
 * - `dist/ShippingManagerCoPilot-v0.1.0/` (portable folder)
 *
 * ### Step-by-Step Build
 *
 * If you want to build components separately:
 *
 * #### Step 1: Compile Python Scripts to .exe
 *
 * ```bash
 * npm run build:python
 * ```
 *
 * This creates:
 * - `dist/get-session-windows.exe`
 * - `dist/login-dialog.exe`
 * - `dist/session-selector.exe`
 *
 * #### Step 2: Compile Node.js Application to .exe
 *
 * ```bash
 * npm run build:node
 * ```
 *
 * This creates:
 * - `dist/ShippingManagerCoPilot.exe`
 *
 * #### Step 3: Package Everything
 *
 * ```bash
 * node build-package.js
 * ```
 *
 * This organizes all files into:
 * ```
 * dist/ShippingManagerCoPilot-v0.1.0/
 * ├── ShippingManagerCoPilot.exe  (main app)
 * ├── helper/
 * │   ├── get-session-windows.exe
 * │   ├── login-dialog.exe
 * │   └── session-selector.exe
 * ├── data/
 * │   └── forecast/  (empty, created at runtime)
 * ├── README.md
 * ├── LICENSE
 * └── START_HERE.txt
 * ```
 *
 * #### Step 4: Create Installer
 *
 * ```bash
 * npm run build:installer
 * ```
 *
 * This creates:
 * - `dist/ShippingManagerCoPilot-Setup-v0.1.0.exe`
 *
 * ## Testing the Build
 *
 * ### Test Portable Version
 *
 * 1. Navigate to `dist/ShippingManagerCoPilot-v0.1.0/`
 * 2. Double-click `ShippingManagerCoPilot.exe`
 * 3. Verify:
 *    - Server starts at https://localhost:12345
 *    - Session extraction works
 *    - Browser opens automatically
 *    - All features functional
 *
 * ### Test Installer
 *
 * 1. Run `dist/ShippingManagerCoPilot-Setup-v0.1.0.exe`
 * 2. Install to default location
 * 3. Launch from Start Menu
 * 4. Verify same functionality as portable version
 *
 * ## Troubleshooting
 *
 * ### PyInstaller Issues
 *
 * **Error: `Module not found`**
 * ```bash
 * # Add missing modules to build-python.spec hiddenimports
 * hiddenimports=['win32crypt', 'win32api', 'cryptography', 'your_missing_module']
 * ```
 *
 * **Error: `Failed to execute script`**
 * - Check Python script runs standalone first: `python helper/get-session-windows.py`
 * - Enable debug mode: Edit .spec file and set `debug=True`
 *
 * ### pkg Issues
 *
 * **Error: `Cannot find module`**
 * - pkg.assets is already configured in package.json (lines 19-38)
 * - Includes: public/, server/, data/forecast/, all dependencies
 * - If adding new dependencies, add them to pkg.assets array
 * - If adding new folders, add them with glob pattern (folder + slash + two asterisks + slash + asterisk)
 *
 * **Error: `Native module not found`**
 * - Some modules (like `keytar`) need native binaries
 * - Solution: Bundle as external dependency or use alternative
 *
 * ### NSIS Issues
 *
 * **Error: `File not found`**
 * - Ensure `npm run build:all` completed successfully
 * - Check paths in installer.nsi match actual file locations
 *
 * **Error: `makensis not found`**
 * - Add NSIS to PATH: `C:\Program Files (x86)\NSIS`
 * - Or use full path: `"C:\Program Files (x86)\NSIS\makensis.exe" installer.nsi`
 *
 * ## File Size Optimization
 *
 * Current estimated sizes:
 * - Python .exe files: ~20-30 MB each (~80 MB total)
 * - Node.js .exe: ~80-100 MB
 * - Total installer: ~180-200 MB
 *
 * ### Reduce Size (Optional)
 *
 * 1. **Enable UPX compression** (in build-python.spec):
 *    ```python
 *    upx=True,
 *    upx_exclude=[],
 *    ```
 *
 * 2. **Exclude unused modules** (in build-python.spec):
 *    ```python
 *    excludes=['tkinter', 'matplotlib', 'numpy'],
 *    ```
 *
 * 3. **pkg compression**:
 *    ```bash
 *    pkg . --compress GZip
 *    ```
 *
 * ## Distribution
 *
 * ### Portable ZIP
 *
 * ```bash
 * cd dist
 * powershell Compress-Archive -Path ShippingManagerCoPilot-v0.1.0 -DestinationPath ShippingManagerCoPilot-v0.1.0-Portable.zip
 * ```
 *
 * ### Installer + Portable Bundle
 *
 * Create both in one release:
 * - `ShippingManagerCoPilot-Setup-v0.1.0.exe` (installer)
 * - `ShippingManagerCoPilot-v0.1.0-Portable.zip` (portable)
 *
 * ## Version Updates
 *
 * 1. Update version in `package.json`
 * 2. Update `APP_VERSION` in `installer.nsi`
 * 3. Rebuild: `npm run build:all && npm run build:installer`
 *
 * ## Clean Build
 *
 * ```bash
 * # Remove all build artifacts
 * rmdir /s /q dist
 * rmdir /s /q build
 * del /q *.spec
 *
 * # Rebuild from scratch
 * npm run build:all
 * npm run build:installer
 * ```
 *
 * ## CI/CD (Future)
 *
 * For automated builds via GitHub Actions:
 * ```yaml
 * - name: Build Application
 *   run: |
 *     npm install -g pkg
 *     pip install pyinstaller
 *     npm run build:all
 *     npm run build:installer
 * ```
 *
 * ## Support
 *
 * If you encounter issues:
 * 1. Check this file for troubleshooting steps
 * 2. Verify all prerequisites are installed
 * 3. Try a clean build
 * 4. Check GitHub Issues for known problems
 */
